# Sample Prompts for School Museum Timeline

Use these in Fooocus prompt box. Mix with your chapter/topic names.

## Prompt Template
`[era], [location], [main subject], [activity], cinematic historical realism, natural textures, detailed environment, dramatic but classroom-safe mood, high detail`

## Negative Prompt Suggestions
- `modern buildings, electric lights, plastic, neon signs, modern clothing, logo, text watermark, blurry face, extra fingers, deformed hands, cartoon style`

## Variation Tips
- **Lighting**: dawn / golden hour / torchlight / moonlight.
- **Lens feel**: wide-angle for context, medium shot for groups, close shot for artifact focus.
- **Camera angle**: eye-level, low angle for dramatic hero feel, overhead for settlement layouts.
- **People count**: 1–2 for focus, 4–8 for community scene.
- **Weather/time**: misty dawn, dry afternoon, windy dusk, clear night.

## Frame-Sequence Prompting Tips (for GIF-like motion)
- Keep prompt mostly fixed; change only one motion cue each frame (for example `flames leaning left` → `upright` → `leaning right`).
- For fire scenes, explicitly vary:
  - flame height/intensity,
  - ember sparkle amount,
  - smoke direction and thickness.
- Keep composition and camera fixed for smoother animation assembly.

## Naming and Export Suggestions
- Use consistent filenames:
  - `event10_fire_f01.png`, `event10_fire_f02.png` ...
  - `event13_caveart_v1.png`, `event13_caveart_v2.png`
- Recommended folder structure:
  - `images/generated/event10_fire/`
  - `images/generated/event13_bhimbetka/`
- Keep a small text note per scene with final prompt + negative prompt used.
